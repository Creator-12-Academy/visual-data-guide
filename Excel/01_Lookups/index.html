Lookup & Reference

The art of finding data. Master the essentials like VLOOKUP and XLOOKUP to connect your datasets dynamically.

V
01. VLOOKUP & HLOOKUP
The classic vertical lookup. Searches the first column and retrieves data from the row.
Essential
X
02. XLOOKUP
The modern replacement. Look left, look right, handle errors, and search arrays easily.
New

03. INDEX & MATCH
The powerful duo. More flexible than VLOOKUP, allowing 2-way lookups and dynamic ranges.
F
04. FILTER Function
Return multiple results based on criteria. The ultimate tool for dynamic lists.
Array
U
05. UNIQUE
Instantly extract a list of distinct values from a range or table.

Sort your data dynamically without touching the 'Sort' button in the ribbon.

Create moving ranges. Useful for scrolling charts and dynamic named ranges.
Adv
N

Turn a text string into a valid cell reference. Great for dynamic sheet references.
C

Select a value from a list based on an index number. Simple but effective.

10. LOOKUP (Legacy)
The original backward-compatible lookup function. Rarely used but good to know.
